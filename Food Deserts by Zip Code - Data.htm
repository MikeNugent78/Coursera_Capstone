Data Sources Income by Zip Code
 

  For income by zip code, this analysis will be using the 2017 income tax file available at www.irs.gov/statistics/soi-tax-stats-individual-income-tax-statistics-2017-zip-code-data-soi. The IRS data contains information by 6 major income groups for each zip code and indicates the individual line item information on U.S. Income Tax forms 1040, 1040A, 1040EZ. The information will be used to create an average income by zip code for correlation and comparative purposes.
